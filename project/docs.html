HTML Documentation

HTML stands for HyperText Markup Language.

HTML is used to create web pages.

It is a markup language, not a programming language.

HTML defines the structure of a webpage.

HTML uses tags to display content.

Tags are written inside angle brackets .

Most tags have opening and closing tags.

HTML elements contain content and tags.

HTML works in all web browsers.

It is the basic building block of the web.

HTML can display text content.

HTML is used to add images to webpages

CSS Documentation.CSS stands for Cascading Style Sheets. 

CSS stands for Cascading Style Sheets.

CSS is used to style web pages.

It controls the appearance of HTML elements.

CSS is not a programming language.

CSS defines colors of text and background.

CSS controls fonts and font sizes.

CSS is used to set margins and padding.

CSS helps in creating layouts.

CSS supports Flexbox layout system.

CSS supports Grid layout system.

CSS makes websites responsive.
